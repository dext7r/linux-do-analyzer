如何开始使用这个工具？

1. 访问 Linux.do 论坛，登录您的账户
2. 进入个人设置页面，找到数据导出功能
3. 下载个人数据包（ZIP 格式）
4. 在本工具中点击"选择文件"，上传您的数据包
5. 等待解析完成，开始查看您的数据分析结果

支持哪些文件格式？

目前支持 Linux.do 论坛导出的 ZIP 格式数据包。数据包内应包含：
• 用户信息文件（JSON 格式）
• 帖子数据文件
• 评论/回复数据文件
• 其他活动记录文件

如何切换主题样式？

在页面右上角找到主题选择器，提供三种主题风格：
• 简洁：清爽简约的苹果风格
• 现代：活力时尚的小米风格
• 商务：专业稳重的华为风格
您的主题选择会自动保存在浏览器中。

数据包很大，处理需要多长时间？

处理时间取决于数据包大小和您的设备性能：
• 小型数据包（<10MB）：通常 10-30 秒
• 中型数据包（10-50MB）：通常 30-120 秒
• 大型数据包（>50MB）：可能需要 2-5 分钟
所有处理都在您的浏览器中进行，不会上传到服务器。

可以分析哪些数据维度？

工具提供多维度数据分析：
• 活动统计：发帖数量、回复数量、点赞收到等
• 时间分析：活跃时段、发帖频率变化趋势
• 内容分析：热门话题、常用词汇、内容分类
• 社交网络：互动对象、影响力分析
• 成长轨迹：等级变化、声誉增长

数据处理失败怎么办？

如果遇到处理失败，请尝试以下步骤：
1. 确认数据包是从 Linux.do 正确导出的
2. 检查数据包是否完整，未损坏
3. 清理浏览器缓存后重新尝试
4. 尝试使用其他现代浏览器（Chrome、Firefox、Safari）
5. 确保设备有足够的内存空间

支持哪些浏览器？

支持所有现代浏览器：
• 推荐：Chrome 88+、Firefox 85+、Safari 14+
• 兼容：Edge 88+、Opera 74+
• 移动端：iOS Safari、Android Chrome
注意：IE 浏览器不受支持，请使用现代浏览器获得最佳体验。

移动设备上可以正常使用吗？

完全支持移动设备：
• 响应式设计，自动适配屏幕尺寸
• 触摸友好的交互界面
• 优化的移动端文件选择体验
• 可以通过云盘应用上传数据文件
建议使用 WiFi 网络处理大型数据包。

为什么图表显示不正常？

图表显示问题可能的原因：
• 网络连接不稳定，CDN 资源加载失败
• 浏览器禁用了 JavaScript
• 广告拦截器阻止了图表库加载
• 浏览器版本过旧，不支持现代 Web 技术
请尝试刷新页面或使用其他网络环境。

我的数据会被上传到服务器吗？

绝对不会！这是我们的核心承诺：
• 100% 客户端处理，所有数据都在您的浏览器中处理
• 没有任何数据上传到我们的服务器
• 没有网络请求发送您的个人信息
• 可以在完全离线的环境下使用
• 开源代码，可以验证我们的承诺

分析结果会被保存在哪里？

分析结果仅保存在您的设备上：
• 使用浏览器的 IndexedDB 本地存储
• 数据不会离开您的设备
• 清理浏览器数据时会被删除
• 其他人无法访问您的分析结果
• 可以随时手动清除所有数据

如何确保数据安全？

我们采用多重安全措施：
• HTTPS 加密传输（仅用于加载页面资源）
• 开源代码，接受社区监督
• 无需注册或登录
• 不收集任何个人标识信息
• 不使用 Cookie 或追踪技术
• 定期安全审计和更新

如何导出分析报告？

提供多种导出选项：
• PDF 报告：完整的分析报告，包含所有图表
• 图片导出：单独保存图表为 PNG 格式
• 数据导出：CSV 格式的原始数据
• 打印友好：优化的打印版本
所有导出都在客户端完成，数据不会上传。
